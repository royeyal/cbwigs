Benefits:

  • Visual Affordance: Users immediately understand there's more content
  • Reduced Confusion: No question about whether content is cut off
  • Improved Engagement: Encourages exploration and interaction

üé® Implementation Details

CSS Mask Implementation:

.swiper-group {
  mask-image: linear-gradient(
    90deg,
    transparent 0%,
    black 8%,
    black 92%,
    transparent 100%
  );
  -webkit-mask-image: linear-gradient(
    90deg,
    transparent 0%,
    black 8%,
    black 92%,
    transparent 100%
  );
}

Customization Variables:

body {
  --swiper-fade-start: 0%;
  --swiper-fade-fade-in: 8%;
  --swiper-fade-fade-out: 92%;
  --swiper-fade-end: 100%;
}

Usage Classes:

  • .swiper-group - Default fade (8%)
  • .swiper-group--subtle-fade - Subtle fade (5%)
  • .swiper-group--strong-fade - Strong fade (12%)
  • .swiper-group--no-fade - Disable fade effect
